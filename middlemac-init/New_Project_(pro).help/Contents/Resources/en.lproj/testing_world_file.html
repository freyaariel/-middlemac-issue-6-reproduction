Testing World

This is another page that uses some of Middlemac’s helpers so that Cucumber testing can be performed. This sample project is also the fixture for testing; if you have this Gem’s source, you can

rake test

…from the Gem’s directory to run the tests yourself.

Note that this isn’t the only file tested, but it’s a nice place to capture some of the test output.

Middlemac configuration information:

partials_dir: Resources/SharedGlobalAssets/_partials
cfBundleIdentifier: com.sample.project.pro.help
cfBundleName: New Project
product_name: New Project Pro
product_version: 3.1.1
product_uri: http://www.sample.com
